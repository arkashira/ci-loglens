# Partner Targets
## Introduction
To further enhance the capabilities of ci-loglens and expand its reach, we have identified key partners for integration. These partnerships will not only increase the tool's functionality but also provide additional revenue streams through affiliate and revenue-sharing models.

## Partner Integration Roadmap
The following SaaS/API integrations are prioritized based on their potential impact, ease of integration, and revenue-sharing opportunities:

1. **CircleCI**
	* Free-tier limits: 1000 pipeline runs per month
	* Integration effort: Medium (M)
	* Value-add: Simplifies CI/CD pipeline monitoring and error identification for developers and DevOps teams
	* Affiliate/revenue-share: Yes, CircleCI has a partner program offering revenue sharing
2. **GitHub Actions**
	* Free-tier limits: 2000 automation minutes per month
	* Integration effort: Small (S)
	* Value-add: Enhances log analysis and error identification for GitHub Actions workflows
	* Affiliate/revenue-share: Yes, GitHub has a partner program with revenue-sharing opportunities
3. **Datadog**
	* Free-tier limits: 1 host, 1 month data retention
	* Integration effort: Large (L)
	* Value-add: Provides comprehensive monitoring and log analysis capabilities for DevOps teams
	* Affiliate/revenue-share: Yes, Datadog has a partner program with revenue-sharing opportunities
4. **New Relic**
	* Free-tier limits: 100,000 events per day
	* Integration effort: Medium (M)
	* Value-add: Offers in-depth application performance monitoring and error analysis
	* Affiliate/revenue-share: Yes, New Relic has a partner program with revenue-sharing opportunities
5. **Splunk**
	* Free-tier limits: 500 MB/day, 1 user
	* Integration effort: Large (L)
	* Value-add: Provides advanced log analysis and security monitoring capabilities for enterprises
	* Affiliate/revenue-share: Yes, Splunk has a partner program with revenue-sharing opportunities
6. **Jenkins**
	* Free-tier limits: Unlimited, open-source
	* Integration effort: Small (S)
	* Value-add: Simplifies log analysis and error identification for Jenkins users
	* Affiliate/revenue-share: No, but a large user base and easy integration make it a valuable partner
7. **GitLab CI/CD**
	* Free-tier limits: 2000 pipeline minutes per month
	* Integration effort: Medium (M)
	* Value-add: Enhances log analysis and error identification for GitLab CI/CD users
	* Affiliate/revenue-share: Yes, GitLab has a partner program with revenue-sharing opportunities

## Prioritization
The prioritization of these integrations is based on the following factors:

* Ease of integration (S/M/L)
* Potential impact on the user base
* Revenue-sharing opportunities
* Alignment with the ci-loglens value proposition

By integrating with these partners, ci-loglens can expand its reach, increase its functionality, and generate additional revenue streams through affiliate and revenue-sharing models.
