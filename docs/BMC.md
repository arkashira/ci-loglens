# Business Model Canvas (BMC.md) for ci-loglens

## Value Proposition
ci-loglens provides AI-powered log analysis and error identification for CI systems, enabling developers and DevOps teams to resolve errors in minutes instead of hours. Key benefits include:
- Automated parsing of complex log formats to extract actionable error messages.
- Real-time visualization of error trends and pipeline health.
- Integration with existing CI tools (GitHub Actions, GitLab CI, Jenkins) to provide a unified debugging experience.
- Reduced debugging time by up to 70
